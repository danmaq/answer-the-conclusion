あとがき

　唐突ですが、私はここ数年、人類を滅ぼすことを夢見ています。
　少々語弊がありました。 命と自我、若さ、そしてマズローの欲求段階説における自己実現欲求を永遠のものとするため、先端科学技術で人工的な不老不死世界、俗称【あの世】を創り、全人類がそこへ移住する、そんな世界を真面目に構想しています。 例えるなら、五億年ボタンの世界に全人類が同居するイメージでしょうか。 映画マトリックスや某忍者漫画の無限月読にも近い話ですね。
　人間をマイクロチップ化、そしてソフトウェア化することで、不老不死を得る。 しかしそれは、人間ではあるが生物ではない──これ、まさに人類を滅ぼす話ですよね。 しかもこの話、恐らく今から三十年そこらで徐々に現実味を帯び始め、さらにそこから十年で実現可能になると思うのです。 ワクワク、ゾクゾクしてきません？

　というわけで本書をご覧の皆様こんにちは、 野村 周平 ( のむら しゅうへい ) です。 同人では『 Ｍｃ ( まく ) 』というペンネームで活動しています。 百ページ超のライトノベルは、恐らくもう二度と描くこともあるまい、と思っていたらあっという間の二冊目です。 これは二度あることはなんとやら、かもしれませんね。 今回の物語は、拙作の東方二次創作ゲーム『 東方夢終劇 ( とうほうむついげき ) 』の後日譚にあたる、スピンオフ ライトノベルです。 夢終劇を世に出したのは実に丁度十年前、サークル活動自体も紆余曲折あって十二年目……長いように見えて、いざ過ぎると、まだ四年か五年前くらいのように感じます。 夢終劇ですが、リリース当初はストーリーなど諸々賛否両論がありました。 時に思い切り核心をつくような考察があったりして、ぎょっとしたのもまた思い出ですね。
　私はサークル立ち上げ以前から持っている信念として、『やったもの勝ち』という考え方があります。 考えるだけ考えぬいて、結局何もしないより、まず何にも考えずバカになりきって踏み出してしまう。 踏み出してからどんなに叩きのめされようが絶対に足を引かず、一寸だけ考えて（ここで考え過ぎてはいけない！）また二歩目を踏み出す。
　数年間社畜になり過ぎて、それすらできていない時期があったりもしましたが、またこの基本に立ち戻っていきたいな、と考えています。 ここまでお読みになった皆さんなら察しが付くかと思いますが、本作は本当に救われないストーリーで、救われる部分を切り取って東方夢終劇として頒布しましたが、それから十年間、残った部分はこれまで誰にも話さず封印して来ました。 ですがこのまま行くと本当に墓場まで持って行きそうな勢いなので『やったもの勝ち』の原点に戻り、今回頒布に踏み切りました。

　さて少々話が変わりますが、この場を借りて一つ大きな発表をしたいと思います。 本作を最後に『東方における大規模二次創作、特にシナリオ』からは引退いたします。 私が作品を創る際、いつも登場キャラになりきり、完全に感情移入した状態でシナリオを練るのですが、それが弱いと例え二次創作であっても良い世界観は作れない、という持論があります。 しかし星蓮船辺りを境目に、世界観が広がりすぎて正直追いかけきれなくなり、感情移入しにくくなった、というのが一番大きいですね。 今後は東方二次創作の活動に関しては、単発でイラストを描いたり、グッズを作ったり、それと既刊の頒布も継続いたしますが、東方のゲーム制作とか今回のような中長編ラノベとか、そういう大規模なことからは、今後引退する方向で考えています。 あ、お誘いを受けたらその限りじゃないですので、ぜひ声かけてくださいね（笑）

　さて、それにしても次回作はどうしたものか。 一応、一つオリジナルな同人ゲームで構想があるのですが、少々規模が大きいうえ、同人というよりも商業向けの要素が強いので、まぁもにょもにょ……。
　それでは長くなってしまったので、今回はこの辺で筆を置きたいと思います。 本書のイラストを手掛けてくださったむつき 灯路 ( ひろ ) さん、そして何よりもここまで本書を読んでくださった皆様に心から御礼申しあげます。 それではまた次回作でお会いしましょう。

　コミケが続く限り、同人活動自体は一生やめない。 た、たぶん　　著・野村　周平（Ｍｃ）
