「……あ」

　真っ白の閃光は次第に和らぎ、その向こうに無機質な天井を映し出す。 天井にはタイルが規則的に敷き詰められ、その合間に空調が埋め込まれ、人工的な風の音を発している。 天井の端からは壁が続き、そしてブラインドが見える。 その隙間から、青い空が覗き込んでいる。 体には純白の布団が覆いかぶさっている。 枕も純白、足元と頭のところに、銀色のパイプが横たわっている。 これは、ベッドの中だ。
　──そっか、夢……だったのか。
　夢、と自覚したことにより、彼女の意識は少しずつはっきりし始めた。 彼女はゆっくりと上半身を起こして、部屋を仕切っているカーテンに手をかける。 そこは小さな病室で、カーテンで仕切られて四つの区画に分かれている。 二つは空席で、カーテンが開いており、置かれているベッドからは掛布団が撤去されている。 残りの一つ……すぐ隣のベッドのカーテンは閉ざされている。 ここには……そうだ、知っている病人がいる。
「あら、おはよう 澪音 ( れいん ) さん」
　不意に開いた病室の扉から、白衣の女性が顔を覗かせて挨拶をしてきた。 この病室を担当している看護婦さんだ。
「あ、おはようございます」
　【 澪音 ( れいん ) 】と呼ばれた彼女も、看護婦さんに挨拶を返す。 言葉を出したせいか、さらに意識が鮮明になり、現実の記憶が徐々に夢の記憶を上書きしていく。
　私の名は玲音、何処にでもいる普通の女子高生……の筈だったが、ひょんなことで大怪我をやらかして入院の身となってしまった。 もうギブスも松葉杖もなしで歩けるようになり、近日中の退院見込みではあるものの、高校の学年末試験に間に合わなくて、いわば「一回休み」の状態だ。 もっと端的にいうと『留年』というやつである。
　不思議とそのことについては嫌な感情が湧いてこない。 寧ろ今までクラスメイトと今ひとつ仲良くできなかった方なので、心機一転出来るのでは、と退院後を楽しみにしているくらいだ。
「今、 涼子 ( りょうこ ) ちゃんは屋上にいますよ、今日は天気いいし、行ってあげたら?」
　先ほどの看護婦の言葉で、 意識から逸れていた隣のベッドの患者のことを思い出した。
　そう、隣のベッドには一つ年下の女の子【 涼子 ( りょうこ ) 】が入院している。 彼女は難病を患っていて、つい先日に目覚めるまで、実に一年近くもの間眠り続けていたらしい。 まだ少ししか話ができていないが、年の近い人は他にいないため新しい友達一号に是非加えたいところ、とチャンスを狙っていたのだ。
